## ✨ Features

### 🖥️ Desktop Environment
- **Hyprland** - Modern Wayland compositor
- **GNOME** - Traditional desktop environment (fallback)
- **GDM** - Display manager

### 🛠️ Development & Tools
- **Editors**: Neovim, Helix, Zed, Sublime Text
- **LSP Servers**: Rust Analyzer, Clang Tools
- **Build Tools**: CMake, GCC, Rustup, Python3
- **Version Control**: Git, GitHub CLI

### 🔒 Security & Privacy
- **VPN**: WireGuard, OpenVPN
- **Privacy Tools**: Tor, Torsocks, OnionShare
- **Firewall**: Strict port configuration
- **Encryption**: Age for secret management

### 🎵 Multimedia
- **Audio**: PipeWire with ALSA and PulseAudio support
- **Music**: MPD with ncmpcpp
- **Video**: MPV with hardware acceleration
- **Visualization**: Cava for audio visualization

### 📦 System Utilities
- **Shell**: Zsh with Oh-My-Zsh and Powerlevel10k
- **Terminal**: Kitty, Alacritty
- **File Sync**: Syncthing
- **System Monitoring**: htop, btop
- **Network Tools**: nmap, aircrack-ng

### 🌐 Browsers & Communication
- **Browsers**: Firefox, Librewolf
- **Messaging**: Telegram Desktop
- **RSS**: Newsboat
